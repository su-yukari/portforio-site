# Portfolio Website

## 概要
オールスクラッチ（VSCode使用）で制作した個人ポートフォリオサイトです。
デザインからコーディングまで一貫して行いました。

## 制作目的
自身のスキルや制作実績を分かりやすく伝えるため。

## 使用技術
- HTML
- CSS
- jQuary
- JavaScript

## 補足
本リポジトリは、ポートフォリオサイトのコード管理用です。

## 工夫した点
情報の優先度を整理し、閲覧者が迷わず目的の情報にたどり着ける構成とレイアウトを意識しました。

## UX視点
「何を見ればよいか」が直感的に分かるよう、導線設計と情報のまとまりを重視しました。
